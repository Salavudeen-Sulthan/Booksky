# Booksky

Booksky is a simple book management web application built using HTML, CSS, and JavaScript.

## Features
- 📚 Add new books
- 📝 Store book title, author, and description
- ❌ Delete books
- 🎨 Clean and responsive user interface
- ⚡ Dynamic DOM manipulation using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes
- DOM Manipulation
- Event Handling
- Form Validation
- Dynamic Element Creation
- Responsive Web Design
